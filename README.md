# StudyBuddy
📚 StudyBuddy
Веб-платформа для автоматического подбора партнёров для совместного изучения учебных дисциплин.

📋 Функциональные возможности
✅ Регистрация и авторизация пользователей (хэширование паролей)
✅ Заполнение профиля с выбором учебных тем и уровней владения
✅ Алгоритм подбора партнёров на основе косинусного сходства (Cosine Similarity) векторов интересов
✅ Система лайков и взаимных совпадений (Matches)
✅ Встроенный мессенджер для совпавших пользователей
✅ База знаний (обмен полезными материалами и ссылками в чате)
✅ Система внутренних уведомлений (о лайках, мэтчах, сообщениях)
✅ Кастомизация интерфейса (Тёмная/Светлая тема, выбор цвета акцентов)

🛠 Технологический стек
Backend: Python 3.10+
Фреймворк интерфейса: Streamlit
База данных: SQLite (встроенная)
Математика: NumPy (для работы с векторами)
Контроль версий: Git / GitHub
🚀 Быстрый старт (Локальный запуск)
Клонировать репозиторий:
git clone https://github.com/valeriya.../StudyBuddy.git
(тут вставь свою ссылку на гитхаб, которую мы брали в прошлом сообщении)
2. Установить зависимости:

bash

pip install -r requirements.txt
Запустить приложение:
bash

streamlit run main.py
Открыть в браузере: http://localhost:8501
📌 Примечание
Демонстрация фронтенд-части (HTML/CSS/JS) доступна в архиве, сданном в Google Classroom. Онлайн-деплой (Streamlit Community Cloud) временно недоступен из-за технических работ на стороне бесплатных облачных провайдеров.
